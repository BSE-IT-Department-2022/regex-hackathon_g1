## Quickstart

1. Clone this repo

```sh
git clone https://github.com/BSE-IT-Department-2022/regex-hackathon_g1.git
```

2. Run the unittest

Command-Line Interface

```sh
python -m unittest -v test_validators.py
```

Test Discovery

```sh
python -m unittest discover -v
```
